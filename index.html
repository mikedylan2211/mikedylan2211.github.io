• Nomad Network

Posted on: December 24, 2025

Official NomadNet GitHub

Nomad Network allows you to build private and resilient communications platforms that are in complete control and ownership of the people that use them. No signups, no agreements, no handover of any data, no permissions and gatekeepers. Nomad Network is build on LXMF and Reticulum, which together provides the cryptographic mesh functionality and peer-to-peer message routing that Nomad Network relies on. This foundation also makes it possible to use the program over a very wide variety of communication mediums, from packet radio to fiber optics.

Nomad Network does not need any connections to the public internet to work. In fact, it doesn't even need an IP or Ethernet network. You can use it entirely over packet radio, LoRa or even serial lines. But if you wish, you can bridge islanded networks over the Internet or private ethernet networks, or you can build networks running completely over the Internet. The choice is yours. Since Nomad Network uses Reticulum, it is efficient enough to run even over extremely low-bandwidth medium, and has been succesfully used over 300bps radio links.

The easiest way to get started using Nomadnet is to download it using a package manager like 'pip'. It can be launched by typing nomadnet in the terminal. The application is entirely terminal based. There is a very good talk from markqvist, the creater of Nomadnet, that explains the underlying network stack called Reticulum. I suggest watching that on Youtube here and reading the official Nomadnet Github linked above.

To get really familiar with Nomadnet and Reticulum, the best way is to install Nomadnet with pip as described above and go through the 'Guide' tab at the top of the screen.

My LXMF is in the Local Peer Info of the screenshot. I host a Pi node, you can add it as a TCP Client Interface.

• MeshCore

Posted on: December 23, 2025

Official MeshCore Website

MeshCore is a multi platform system for enabling secure text based communications utilising LoRa radio hardware. It can be used for Off-Grid Communication, Emergency Response & Disaster Recovery, Outdoor Activities, Tactical Security including law enforcement and private security and also IoT sensor networks.

MeshCore is free and open source:

  • MeshCore is the routing and firmware, available on GitHub under the MIT license
  • There are clients made by the community, such as web clients; these are free to use, and some are open source
  • The cross-platform mobile app developed by Liam Cottle for Android, iOS, and PC is free to download and use
  • The T-Deck firmware, developed by Scott at Ripple Radios (the creator of MeshCore), is free to flash on your devices and use

I currently run a repeater in the attic and am working on a solar roof node.

In the official Meshcore Github in the meshcore_py repo there is a serial_pingbot.py and many more cool python templates (https://github.com/meshcore-dev/meshcore_py/tree/main/examples). I needed a pingbot that works over BLE so I adapted the script to work over BLE. It's in my Github.
